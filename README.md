## About

**👨‍💻 Frontend Developer (Upwork) · 🎓 Computer Science Student · 🔐 SOC Analyst (aspiring)**

Currently: freelancing on Upwork • studying Meta Frontend cert • learning cybersecurity fundamentals.

- Small automation scripts and dev tools
